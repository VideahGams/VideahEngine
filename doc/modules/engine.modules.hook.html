Module engine.modules.hook

The hook library allows scripts to interact with game or user created events.

It allows you to "hook" a function onto an event created with hook.run or hook.call and run code when that events happens that either works independently or modifies the event's arguments. This is the preferred method instead of overriding functions to add your own code into it.
